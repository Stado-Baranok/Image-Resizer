🖼️ Image Resizer – Ресайзер изображений 
Мощный инструмент для изменения размера изображений с поддержкой пакетной обработки, рекурсивного обхода папок, сохранения пропорций и выбора алгоритма интерполяции.
Поддерживает PNG, JPG, BMP, TIFF, WEBP и другие форматы.
Реализован на 7 языках программирования с единым интерфейсом командной строки.

🚀 Возможности
Изменение размера – задайте ширину и высоту в пикселях или процентное соотношение.

Сохранение пропорций – автоматическое вычисление второй стороны для сохранения соотношения сторон.

Режимы изменения:

fit – вписать изображение в заданные размеры (сохраняя пропорции).

fill – заполнить область, обрезая лишнее (центрирование).

stretch – растянуть до точных размеров (без сохранения пропорций).

Пакетная обработка – обработайте все изображения в папке за один раз.

Рекурсивный обход – обработайте подпапки (-r).

Настройка качества – для JPG и WEBP (опция -q).

Поддержка форматов – на входе и выходе.

Прогресс-бар – наглядный индикатор выполнения.

Цветной вывод – информативные сообщения.

Кроссплатформенность – Windows, Linux, macOS.

📖 Использование
Синтаксис (единый для всех версий):

bash
<команда> <вход> [выход] [опции]
Аргументы
Аргумент	Описание
<вход>	Входной файл или папка с изображениями
[выход]	Выходной файл или папка (если не указан, используется тот же путь)
Опции
Опция	Описание
-W, --width <N>	Ширина в пикселях
-H, --height <N>	Высота в пикселях
-p, --percent <N>	Процент от исходного размера (например, 50)
-m, --mode <fit|fill|stretch>	Режим изменения (по умолчанию fit)
-q, --quality <N>	Качество JPG/WEBP (1–100, по умолчанию 90)
-r, --recursive	Рекурсивно обрабатывать подпапки
-f, --format <ext>	Выходной формат (по умолчанию сохраняется исходный)
-v, --verbose	Подробный вывод
-h, --help	Справка
Примеры
bash
# Изменить размер одного изображения до 800x600 (с сохранением пропорций)
python resizer.py photo.jpg -W 800 -H 600

# Уменьшить все изображения в папке на 50%
python resizer.py ./images ./resized -p 50

# Заполнить область 1024x768 с обрезкой (режим fill)
python resizer.py input.png output.png -W 1024 -H 768 -m fill

# Рекурсивно обработать все изображения с качеством 85
python resizer.py ./photos ./output -r -q 85

🧠 Логика работы
Сбор файлов – сканирование входной папки (рекурсивно при -r) или использование одного файла.

Фильтрация – только изображения с поддерживаемыми расширениями.

Загрузка – чтение изображения в память.

Изменение размера – в зависимости от режима (fit, fill, stretch) и заданных размеров/процента.

Сохранение – запись в выходной файл с сохранением структуры папок (при рекурсивном режиме).

Прогресс-бар – отображение хода обработки.

✨ Дополнительные фичи
Автоопределение формата – выходной формат определяется по расширению выходного файла.

Сохранение структуры папок – при рекурсивном обходе структура повторяется.

Обработка ошибок – программа продолжает работу при ошибках отдельных файлов.

Многопоточность – в некоторых реализациях для ускорения.
